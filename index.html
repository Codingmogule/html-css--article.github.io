LEARN HTML.....

What is HTML??

HTML stands for HyperText Markup Language.
It's the standard markup language used for creating web pages.
It defines the structure and layout for a website's content using tags.
Tags mostly come in pairs i.e Opening and Closing tags but some only have opening tags e.g the image tag.

Why use HTML??

HTML allows one to present content on web pages inform of texts,images,videos...
It also allows one to embed content from other sites(a website inside a website king of thing)
HTML allows one to create clickable links that connect web pages togeher.
It basically defines the logical structure of a web.
To sum it up,it's basically the best there isand it's universally recognized!

How to use HTML!!

HTML is the most basic building block that acts as a backbone of webpages.
We use the DOM concept.Wondering what this is???I got you covered.
......
DOM stands for Document Object Model.
It is the interface for the browser and is structured like a tree.
Each element is represented as an object with properties and methods that can be styled using CSS or manipulated using JavaScript
Key Concepts in DOM
1.Doc Object - Document comes first in hierarchy.It represents the entire HTML document and provides methods to access the elements within the document.
2.Elements - HTML has lots of elements a few being div tags,paragraph tags,image tags,anchor tags and they are represented as objects which have properties that allow you to modify attributes,content and style elements.
You can find more elements on W3SCHOOLS
3.Nodes - They represent different parts of the HTML document i.e
.Element nodes
.Text nodes
.Comment nodes...

LEARN CSS.....

What is CSS??

CSS stands for Cascading Style Sheets.
It is a markup language that is used to add style to web pages.
It contains selectors such as:
.Element Selectors.
.Class Selectors.
.ID Selectors.
.Pseudo Class Selectors.

Why use CSS??

CSS makes you webpage visually appealling.
Analogy.....It is like painting a house after you've built it using HTML.
CSS also helps in responsiveness by making the webpage compatible with on different devices.

How to use CSS!!

.First you start by linking HTML to CSS at the head section using the link tag.
.Then select a HTML element using selectors to style it.
.Apply CSS styles using CSS properties e.g color,font,padding,margin...Then experiment and refine.
